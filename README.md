# App Finanças Pessoais Conversacional

Aplicativo de organização de finanças pessoais com interação via conversas.  
O objetivo é simplificar o controle financeiro, tornando-o mais natural e acessível para iniciantes.

---

## Contexto
Muitas pessoas desistem de controlar seus gastos porque os apps atuais exigem muita entrada manual e pouca personalização.  
Este projeto busca resolver isso com uma experiência baseada em chat e recomendações automáticas de economia.

---

## Problema
- **Dificuldade atual:** Apps de finanças exigem entradas manuais extensas e oferecem pouca personalização.  
- **Impacto:** Usuários iniciantes desistem rapidamente do controle financeiro por acharem o processo burocrático.  
- **Oportunidade:** Criar uma experiência fluida e personalizada, baseada em conversas e recomendações automáticas.

---

## Público-Alvo
- Pessoas que desejam começar a organizar suas finanças sem complexidade.  
- Usuários iniciantes que não têm familiaridade com planilhas ou apps tradicionais.  
- Jovens adultos e profissionais que buscam praticidade no dia a dia.  

---

## Funcionalidades-Chave
1. Registro de gastos via chat em linguagem natural.  
2. Classificação automática das transações.  
3. Definição e acompanhamento de metas financeiras.  
4. Dicas de economia enviadas pelo “Agente Financeiro”.  
5. Relatórios simples e personalizados.  

---

## MVP
- **Principais telas:**
  - Tela de chat (interação com o assistente).  
  - Tela de metas (definição e acompanhamento).  
  - Tela de relatórios (gráficos simples e insights).  

- **Recursos necessários:**
  - Processamento de linguagem natural (NLP).  
  - Motor de categorização automática.  
  - Sistema de notificações e recomendações.  
  - Banco de dados para armazenar transações e metas.  

- **Validação inicial:**
  - Testes com grupo piloto de usuários iniciantes.  
  - Medir engajamento (quantidade de interações no chat).  
  - Avaliar clareza dos relatórios e utilidade das dicas.  

---

## Tecnologias
- Stack web moderna (React, Vite, Node).  
- Banco de dados para armazenamento de transações e metas.  
- Processamento de linguagem natural (NLP).
